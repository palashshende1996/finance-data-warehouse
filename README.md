# Finance Data Warehouse & CFO Dashboard

## Objective
Build an end-to-end finance data warehouse using SQL Server
with automated ETL pipelines and Power BI dashboards.

## Tech Stack
- SQL Server
- SQL Server Management Studio (SSMS)
- Power BI
- GitHub

## Data Model
- Fact_GL
- Fact_AR
- Fact_AP
- Dim_Date
- Dim_Customer
- Dim_Vendor
- Dim_Department
